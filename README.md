# Project Portfolio

Source for my GitHub Pages portfolio site, published at https://spicytenderloin.github.io/projects/

## Adding a new project

1. Create an images folder: `assets/img/<project-slug>/`
2. Create a page: `projects/<project-slug>/index.html` (copy an existing one as a template)
3. Add a card linking to it in [index.html](index.html)
